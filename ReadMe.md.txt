# Transleatha - Cross-Border Digital Trade Platform

**Connecting Nigeria, Niger, and Mali through smarter trade technology.**

---

## 🌍 About Transleatha

Transleatha is a web-based platform designed to simplify and empower cross-border trade in West Africa.  
It focuses on key trade challenges like:

- Customs document uploads
- Real-time logistics tracking
- A secure digital wallet
- A vibrant marketplace for African goods
- A multilingual help center chatbot (English, French, Hausa)
- Admin backend for managing documents and products

Built to support the goals of the **AfCFTA Digital Trade Protocol**.

## 🛠️ Features

- 🌐 Navigation across Marketplace, Wallet, Customs, Logistics, Help Center
- 🛒 Digital Marketplace for leather and agricultural products
- 📜 Upload customs documents (Export Permit, Form M, Phytosanitary Certificate)
- 🚚 Track shipments from Nigeria to Mali with status updates
- 💬 Multilingual chatbot support
- 👨‍💼 Admin dashboard to approve documents and add new products
- 🔐 Admin login and logout system
